「ターミナル」の日本語パスが文字化け対策

「ターミナル」の日本語パスが文字化けする

  1. .profile edit
     $ vi ~/.profile
  2. .profile write
     export LANG=ja_JP.UTF-8   
 export LESSCHARSET=utf-8
  3. 設定反映
     $ source ~/.profile
